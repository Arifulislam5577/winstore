Finsweet is a fully mobile responsive podcasting website built with React Js and the most popular CSS framework Tailwind CSS.

There are three pages here.

1. Home page
2. About Page
3. 404 Page

### Used Technology

- React Js
- Tailwind CSS
- React Router Dom
